# React Application to search all your robot friends!
Tutorial for udemy course - React
To run the project:

1. Clone this repo
2. Run `npm install`
3. Run `npm start`


# Technologies Used
* HTML5
* CSS3
* React.js
* Redux
## Packages
* Create React App
* Tachyons(stlying packing)
* Redux-thunk
* Redux-logger
## APIs
* RoboHash 
* JSONPlaceholder
