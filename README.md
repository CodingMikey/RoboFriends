# React Application to search all your robot friends!
Tutorial for udemy course - React
To run the project:

1. Clone this repo
2. Run `npm install`
3. Run `npm start`


# Technologies Used
## Front-End
* HTML5
* CSS3
* React.js
## Packages
* Create React App
* Tachyons(stlying packing)
## APIs
* RoboHash 
* JSONPlaceholder
